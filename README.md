# Run Tracker (Clean Build)

**Version:** V47
**Status:** L.ve
            
## Description
A lightweight web application that tracks GPS coordinates with UI for live recording images/video clips. 
Designed to run directly in a mobile browser without installation. So, no cockies.

## Features
* **Hybrid Recording:** Captures images/ Records video and GPS location simultaneously.
* **Auto-time-split recordings:** Videos are chunked at recrding time into k secs and saved automatically (prevents memory crashes).
* **Auto-List Storage:** Images/Videos are saved automatically to IndexedDB (prevents memory crashes).
* **Smart Export:** Downloads JSON data linked to video filenames.
